# Offline Text Summarization Engine (C++)

An offline backend system that summarizes large text documents using an
extractive, frequency-based approach without using any cloud APIs.

## Problem Statement
Most text summarization tools rely on internet connectivity. This project
works completely offline, ensuring data privacy and usability in
low-connectivity environments.

## Features
- Fully offline processing
- Frequency-based extractive summarization
- Lightweight and efficient
- Written in pure C++

## Tech Stack
- C++
- STL
- File Handling
- OOP Concepts

## How to Run
1. Place text inside `input.txt`
2. Compile: `g++ main.cpp -o summarizer`
3. Run: `./summarizer`
